# Vita Salud

![Vita Salud Logo](icon.png)

**Vita Salud** es una aplicación móvil de seguimiento de salud y nutrición diseñada por profesionales sanitarios. Permite registrar hábitos diarios, acceder a contenido educativo respaldado por evidencia científica y personalizar la experiencia según el perfil de salud del usuario.

Desarrollada por **Lucas Galdino** y **Deyanira Navarro** — enfermeros con experiencia clínica en nutrición y salud integral.
